### 环境需要
node v12.6.0
npm v6.9.0
### 首次运行
```
### 安装node-sass 
npm install node-sass@6.0.1 --registry=https://registry.npmmirror.com --sass_binary_site=https://npmmirror.com/mirrors/node-sass/ -D

### 安装依赖
npm i
```
### 项目启动
```
npm i
npm start
```
